# breaks: Interactive Watershed Break Point Delineation

Interactive Shiny app for clicking stream networks to place break
points, delineating upstream watersheds, and exporting sub-basins via
pairwise subtraction. Built on the fresh package for FWA-referenced
spatial hydrology queries.
